**Important Notes for Glyphs 3 Users:** These lecture videos show demos utilizing Glyphs 2, which is the older version of the [Glyphs 3 App](https://glyphsapp.com/). However, as these demos do not use any of the functions unique to Glyphs 3, anyone following along with Glyphs 3 should not have a problem.

When using Glyphs 3, after launching the software, click on New Document -> Latin -> Create Document. Then your screen should look similar to any of the demos below!
